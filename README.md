# Wix Producs Catalog
Wix Producs Catalog


## Getting Started

Make sure to have the environment variables set up correctly, you can copy the values from .env.example with the
following command:

```bash
cp .env.template .env
```

Run the development server:

```bash
docker-compose up
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
